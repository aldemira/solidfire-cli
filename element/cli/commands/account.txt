                  </row>
               </thead>
               <tbody cid="7uqUX">
                  <row cid="2EBjiH">
                     <entry colname="col1" cid="1Thqa5">
                        <apiname>ModifyAccount</apiname> enables you to modify an existing account. When you lock an account, any existing connections from that account are immediately terminated. When you change an account's CHAP settings, any existing connections remain active, and the new CHAP settings are used on subsequent connections or reconnections. To clear an account's <parmname>attributes</parmname, specify {} for the <parmname>attributes</parmname parameter. 
                     </entry>
                     <entry colname="col2" cid="sKwWv">
                        <parmname>accountid</parmname>
                        : Specifies the AccountID for the account to be modified. 
                        <parmname>username</parmname>
                        : Specifies the <parmname>username</parmname associated with the account. (Might be 1 to 64 characters in length). 
                        <parmname>status</parmname>
                        : Sets the <parmname>status</parmname for the account. Possible values are: active: The account is active and connections are allowed. locked: The account is locked and connections are refused. 
                        <parmname>initiatorsecret</parmname>
                        : Specifies the CHAP secret to use for the initiator. This secret must be 12-16 characters in length and should be impenetrable. The initiator CHAP secret must be unique and cannot be the same as the target CHAP secret. 
                        <parmname>targetsecret</parmname>
                        : Specifies the CHAP secret to use for the target (mutual CHAP authentication). This secret must be 12-16 characters in length and should be impenetrable. The target CHAP secret must be unique and cannot be the same as the initiator CHAP secret. 
                        <parmname>attributes</parmname>
                        : List of name-value pairs in JSON object format. 
                     </entry>
                  </row>
               </tbody>
            </tgroup>
         </table>
