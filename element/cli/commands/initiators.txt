                  <row cid="2EBjiH">
                     <entry colname="col1" cid="1Thqa5">
                        <apiname>ModifyInitiators</apiname> enables you to change the attributes of one or more existing <parmname>initiators</parmname. You cannot change the name of an existing initiator. If you need to change the name of an initiator, delete it first with DeleteInitiators and create a new one with CreateInitiators. If <apiname>ModifyInitiators</apiname> fails to change one of the <parmname>initiators</parmname provided in the parameter, the method returns an error and does not modify any <parmname>initiators</parmname (no partial completion is possible). 
                     </entry>
                     <entry colname="col2" cid="sKwWv">
                        <parmname>initiators</parmname>
                        : A list of objects containing characteristics of each initiator to modify. Values are: initiatorID: (Required) The ID of the initiator to modify. (Integer) alias: (Optional) A new friendly name to assign to the initiator. (String) attributes: (Optional) A new set of JSON attributes to assign to the initiator. (JSON Object) volumeAccessGroupID: (Optional) The ID of the volume access group into to which the initiator should be added. If the initiator was previously in a different volume access group, it is removed from the old volume access group. If this key is present but null, the initiator is removed from its current volume access group, but not placed in any new volume access group. (Integer) 
                     </entry>
                  </row>
               </tbody>
            </tgroup>
         </table>

                     <entry colname="col1" cid="1Thqa5">
                        <apiname>CreateInitiators</apiname> enables you to create multiple new initiator IQNs or World Wide Port Names (WWPNs) and optionally assign them aliases and attributes. When you use <apiname>CreateInitiators</apiname> to create new <parmname>initiators</parmname, you can also add them to volume access groups. If <apiname>CreateInitiators</apiname> fails to create one of the <parmname>initiators</parmname provided in the parameter, the method returns an error and does not create any <parmname>initiators</parmname (no partial completion is possible). 
                     </entry>
                     <entry colname="col2" cid="sKwWv">
                        <parmname>initiators</parmname>
                        : A list of objects containing characteristics of each new initiator. Values are: name: (Required) The name of the initiator (IQN or WWPN) to create. (String) alias: (Optional) The friendly name to assign to this initiator. (String) attributes: (Optional) A set of JSON attributes to assign to this initiator. (JSON Object) volumeAccessGroupID: (Optional) The ID of the volume access group into to which this newly created initiator will be added. (Integer) 
                     </entry>
                  </row>
               </tbody>
            </tgroup>
